# 파비콘(Favicon) 가이드

이 프로젝트는 다음과 같은 파비콘 이미지가 필요합니다:

1. `favicon-16x16.png` - 16x16 픽셀
2. `favicon-32x32.png` - 32x32 픽셀
3. `apple-touch-icon.png` - 180x180 픽셀
4. `android-chrome-192x192.png` - 192x192 픽셀
5. `android-chrome-512x512.png` - 512x512 픽셀

이 파일들을 `/public` 디렉토리에 추가해주세요.

## 파비콘 생성 도구

다음 도구를 사용하여 파비콘을 생성할 수 있습니다:
- [Favicon Generator](https://realfavicongenerator.net/)
- [Favicon.io](https://favicon.io/)
