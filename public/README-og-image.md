# OpenGraph 이미지 가이드

소셜 미디어에서 공유 시 보여질 OpenGraph 이미지를 만들어 `/public` 디렉토리에 추가해주세요.

## 필요한 이미지

1. `og-image.png` - 1200x630 픽셀 크기의 이미지

## 이미지 포함 내용 제안

- 포트폴리오 사이트 제목
- 간단한 설명 또는 슬로건
- 전문 분야 (예: "Frontend Developer")
- 선명한 배경색 및/또는 브랜딩 요소

## 디자인 도구

다음 도구를 사용하여 OpenGraph 이미지를 만들 수 있습니다:
- Figma
- Adobe Photoshop
- Canva (https://www.canva.com)
- Pixlr (https://pixlr.com)

## 크기 및 형식 팁

- 권장 크기: 1200x630 픽셀 (최소 600x315 픽셀)
- 주요 콘텐츠는 가운데에 배치하고, 이미지 양쪽 끝에 여백을 둡니다.
- PNG 형식을 권장하지만, JPG도 괜찮습니다.
- 파일 크기를 최적화하세요 (300KB 미만 권장).
